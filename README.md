## Класс Utils

Файл `Utils.cs` содержит класс с двумя методами:

```csharp
public class Utils {
    public void Print() => Console.WriteLine("Utils");
    public int Add(int a, int b) => a + b;
}
```

| Метод | Описание |
|-------|----------|
| `Print()` | Выводит строку "Utils" в консоль |
| `Add(int a, int b)` | Возвращает сумму двух целых чисел |

## История коммитов

```
* <хеш> (HEAD -> main, origin/main, utils) Add function Add
* <хеш> Add Utils.cs
```

Все коммиты линейны благодаря fast-forward слиянию.

## Инструкция по воспроизведению

### Шаг 1: Создание локального репозитория и первого коммита

Откройте терминал в папке `UtilsProject`:

```bash
git init
git add Utils.cs
git commit -m "Add Utils.cs"
```

### Шаг 2: Создание ветки `utils` и добавление новой функции

```bash
git switch -c utils
# Добавьте метод Add в Utils.cs
git add Utils.cs
git commit -m "Add function Add"
```

### Шаг 3: Слияние ветки `utils` в `main`

```bash
git switch main
git merge utils  # fast-forward слияние
```

### Шаг 4: Отправка в удалённый репозиторий через Visual Studio

1. Откройте проект в Visual Studio
2. В **Team Explorer** → **Settings** → **Repository Settings** → **Remotes** → **Add**
3. Укажите `origin` и URL удалённого репозитория
4. В разделе **Sync** нажмите **Push**
